#  OOP Projects in C++

A collection of 5 Object-Oriented Programming projects built in C++ as part of university coursework at UMT Lahore.

##  Projects

| # | Project | Features |
|---|---------|----------|
| 1 | Citizen Management | Add & view citizens with ID, name, age, city |
| 2 | Hospital Management | Admit, discharge & view patients |
| 3 | Library Management | Issue, return & view books |
| 4 | Student Management | Add & view student records |
| 5 | Hotel Management | Book, free & view rooms |

##  Concepts Used
- Classes and Objects
- Encapsulation (private/public)
- STL Vector
- Menu-driven programs (do-while loop)

##  How to Run Any Project
```bash
g++ main.cpp -o output
./output
```
